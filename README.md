# Implementasi Algoritma Sorting

Program sorting dalam **Python** dan **C++** yang mengimplementasikan 5 algoritma sorting dengan menampilkan proses setiap iterasi.

## Algoritma yang Diimplementasikan
- Bubble Sort
- Selection Sort
- Insertion Sort
- Quick Sort
- Merge Sort

## Input & Output

**Input:**
- Jumlah data
- Data yang akan diurutkan (dimasukkan satu per satu)

**Output:**
1. Tampilan data pada setiap iterasi/langkah
2. Data hasil sorting

## Cara Menjalankan

### Python
```bash
python sorting.py
```

### C++
```bash
g++ -o sorting sorting.cpp
./sorting
```

## Contoh Output (Python / C++)

```

Data awal: [64, 25, 12, 22, 11]

── Bubble Sort ──

  Iterasi 2: [12, 22, 11, 25, 64]
  Iterasi 3: [12, 11, 22, 25, 64]
  Iterasi 4: [11, 12, 22, 25, 64]
  Hasil: [11, 12, 22, 25, 64]
```
